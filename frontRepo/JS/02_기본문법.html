기본문법

1. 변수와 자료형

1) 변수 선언

        지역변수, 전역변수
        var , let , const

2) 자료형

        JS에서는 자료형을 지정하지 않고, 리터럴에 의해 타입이 결정된다.
  • string
  • number
  • boolean
  • object
  • function
  • undefined

2. 데이터 형변환

1) 문자열과 숫자의 + 연산

        문자열 + 숫자 = 문자열
        숫자 + 숫자 + 문자열 = 문자열 (앞의 2개 숫자가 먼저 연산되고, 뒤의 문자열과 연산됨)

2) 강제 형변환

        문자형 -> 숫자형
        Number , parseInt() , parseFloat() 

        숫자형 -> 문자형
        String

3. 연산자

== vs ===

        == : 값의 일치 여부
        === : 값 && 타입 일치여부
