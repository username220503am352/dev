BOM (Browser Object Model)

window 객체

        - JS 최상위 객체
        - 브라우저 내 모든 요소, 모든 객체 를 포함한다.
        BOM : location , navigator , history , screen
        DOM : document

1) window.open

2) 타이머


        setTimeout  : 타이머 종료 시, 한번 실행
        setInterval : 타이머가 반복되며 함수 실행

        clearTimeout : 타이머를 없앰
        clearInterval: 타이머를 없앰

3) location

        주소표시줄 관련 객체

4) history

        방문 기록 관리하는 객체

5) navigator

        브라우저 정보를 관리하는 객체

6) screen

        (OS의 화면)모니터의 정보를 담고있는 객체
