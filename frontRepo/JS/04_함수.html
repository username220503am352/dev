1. 함수 선언

1) 선언적 함수

        function 키워드에 함수 작성

2) 익명 함수

        함수명 작성 X
        (변수에 대입)

3) 자동 실행 함수

        함수를 정의하는 동시에 실행
        한번만 호출 (함수 이름이 없으므로 다시 호출할 수 없음)

4) 화살표 함수

        ES6 추가됨

        * 파라미터 부분 변형 가능
        - 매개변수 0개      : () => {~~~}
        - 매개변수 1개      :  x => {~~~}
        - 매개변수 2개 이상 :  (x,y) => {~~~}

        * 리턴 처리
        - 로직이 여러 줄 일 때 : {~~~} 중괄호 필요
        - 로직이   한 줄 일 때 : (..) => return 리턴값;
        - 로직이   한 줄 일 때 : (..) => 리턴값;
