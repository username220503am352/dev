정규표현식

        정규표현식 (Regular Expression)
        특정 규칙을 가진 문자열 검색에 사용

1. 정규표현식 객체 생성

        생성자 함수     : RegExp 
        리터럴          : /정규표현식/플래그

2. 플래그 문자

        정규표현식 검색 방식을 설정하기위해 사용
        여러개 동시에 사용 가능 (순서 상관 X)
  • i : 대소문자 무시하고 검색
  • g : 전역으로 비교
  • m : 여러 줄

3. 메타문자

        메타문자로 특정 패턴을 제시할 수 있음
