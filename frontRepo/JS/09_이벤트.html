이벤트

1. 이벤트 모델의 종류

1) 고전 이벤트 모델

        이벤트 속성에 핸들러 연결
        (삭제하려면 null 할당)

2) 인라인 이벤트 모델

        태그 요소 내부에 작성
        - 동작할 내용을 직접 작성
        - 선언된 함수를 호출

3) 표준 이벤트 모델

        w3 에서 공식적으로 지정한 이벤트 모델
        한번에 여러개의 핸들러 설정 가능

2. 이벤트가 발생한 요소 객체에 접근

        이벤트가 발생한 요소 객체는
        내부적으로 이벤트 발생 정보를 담고 있는 객체를 매개변수로 전달

        이벤트 객체를 통해서 요소에 접근 가능

        콜백함수의 종류에 따라 this 로 접근 가능
